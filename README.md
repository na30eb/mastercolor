# Master Color

Master Color is a lightweight Chrome extension designed for designers, content creators, developers, and digital professionals who need a fast and elegant way to pick colors, generate palettes, and manage color workflows.

## Features

- Color Picker
- HEX & RGB Copy
- Color History
- Palette Generator
- Lightweight & Fast

## Installation

1. Download the repository.
2. Open Chrome Extensions.
3. Enable Developer Mode.
4. Click Load Unpacked.
5. Select the extension folder.

## مستر کالر چیست؟

Master Color یک افزونه سبک برای مرورگر Chrome است که به طراحان، تولیدکنندگان محتوا، توسعه‌دهندگان وب و صاحبان کسب‌وکار کمک می‌کند رنگ‌ها را سریع‌تر انتخاب، ذخیره و مدیریت کنند.

### 🎨 انتخاب رنگ

استخراج رنگ از صفحات وب.

کپی مستقیم کدهای:

- HEX
- RGB

ذخیره آخرین رنگ‌های انتخاب‌شده.
